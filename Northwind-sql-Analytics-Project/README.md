# Northwind Insights SQL Project (MySQL)


## Project Highlights

- Clean, modular SQL scripts by difficulty and business use case.
- queries includes(joins, CTEs, window functions, subqueries, aggregates).
- Reusable views and stored procedures for reporting.
- KPI-driven analysis: sales trends, top customers, employee performance, and shipping efficiency.

## Tech Stack

- MySQL 8+
- Northwind sample database
- SQL only (no external dependencies)
